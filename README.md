# 🎬 Movie Recommendation System

A Machine Learning-based Movie Recommendation System developed using Python, Pandas, Scikit-Learn, and Streamlit.

## Overview

This project recommends movies based on content similarity by analyzing movie metadata such as genres, keywords, cast, crew, and overview. The system uses text vectorization and cosine similarity to identify and recommend similar movies.

## Features

* Content-based movie recommendations
* Feature engineering using movie metadata
* CountVectorizer for text vectorization
* Cosine Similarity for recommendation generation
* Interactive Streamlit web application

## Dataset

TMDB 5000 Movies Dataset

* 4,800+ movies
* Genres, cast, crew, keywords, and overview data
